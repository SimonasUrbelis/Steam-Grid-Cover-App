Set cover art for the new Steam library.

App made by T1lt3d

In order for the app to create its own game cover when it can't find one, make sure to download and install GM from http://www.graphicsmagick.org/

You can get your Steam API key steamcommunity.com/dev/apikey

You can get your SteamID64 and SteamID3 from steamid.io

Your userdata folder is usually under C:\Program Files (x86)\Steam\userdata\[Your SteamID3]

If you are getting errors or have a slow PC, try lowering download speed
